# catalog-app
# Dynamic Multi-Category Catalog

## 🚀 Tech Stack
- React.js
- React Router
- CSS

## 📌 Features
- Category-wise product listing (Cars, Bikes, Phones, Computers)
- Dynamic item detail page
- Properties rendered dynamically using JSON (itemprops)
- Responsive grid layout
- Navigation between pages

## 🧠 Approach
- Parsed the given JSON data
- Grouped items by category using JavaScript
- Built reusable components (CategorySection, ItemCard)
- Implemented routing using React Router
- Dynamically rendered properties using map()
